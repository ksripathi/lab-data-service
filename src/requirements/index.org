* Introduction 

- Lab spec has to be checked into the github repository with
  developer, lab, exp, inst and discipline details in =json= file.

- When user made changes to Github repository a =web-hook= will invoke =lab-data-service's= API which will then clone the repository
  and read the changed lab specs and store them into database.

- All lab specs in github repository all readonly
