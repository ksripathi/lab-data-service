  This document describes all the requirements of =lab-data-service=.
* Requirements
  :PROPERTIES:
  :CUSTOM_ID: requirements
  :END: 
** Req1
  :PROPERTIES:
  :CUSTOM_ID: req1
  :END: 
   - Able to store the data about labs using lab specification files. 
** Req2
  :PROPERTIES:
  :CUSTOM_ID: req2
  :END: 
   - On updation of labspecification files, data of lab should be
     automatically updated on database.
** Req3 
  :PROPERTIES:
  :CUSTOM_ID: req3
  :END: 
   - Able to query data about the labs. Able to search all labs by lab name.
** Req4
  :PROPERTIES:
  :CUSTOM_ID: req4
  :END: 
   - Able to get resource ids of section/sub-sections of a given lab.

** Req5
  :PROPERTIES:
  :CUSTOM_ID: req5
  :END: 
  - Footer need to be served from Lab Data Service. All the portals like
    OpenedX, Outreach Portal, and Feedback should be able to use the footer.
** Req6
   :PROPERTIES:
   :CUSTOM_ID: req6
   :END:
  - Able to provide Hosting Information.
** Req7 
   :PROPERTIES:
   :CUSTOM_ID: req7
   :END:
  - Able to provide Lab Assets.
** Req8
   :PROPERTIES:
   :CUSTOM_ID: req8
   :END:
  - Able to provide the Lab Integration status.
