* Introduction 
  This document describes all the requirements for =lab-data-service=.

* Requirements
** Req1
   - Able to store the data about labs using lab specification files. 
** Req2 
   - Able to query data about the labs.
** Req3
   - Able to get resource ids of section/sub-sections of a given lab.
